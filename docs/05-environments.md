# 05. Среды

Принципы одинаковы везде. Различаются только интерфейсы, горячие клавиши и названия режимов. Ниже — краткие карточки для популярных сред.

---

## Cursor

**Особенности:**
- Режим **Chat** — общение с агентом в боковой панели.
- Режим **Composer** — агент может редактировать несколько файлов и запускать команды.
- **Agent mode** — агент сам вызывает инструменты (поиск, терминал, правки).
- `.cursorrules` — файл с правилами поведения агента в проекте.

**Как начать:**
1. Открой проект в Cursor.
2. Нажми `Cmd/Ctrl + L` — откроется чат.
3. Опиши задачу и попроси план.
4. В режиме Agent дай разрешение на команды.

**Что помнить:** Cursor любит делать много изменений сразу. Контролируй scope.

---

## Kimi Code

**Особенности:**
- Работает в терминале и IDE.
- Поддерживает **skills** — переиспользуемые инструкции.
- Читает `AGENTS.md` в корне проекта.
- Имеет доступ к bash, файлам, поиску.

**Как начать:**
1. Установи Kimi Code CLI.
2. Зайди в папку проекта.
3. Создай `AGENTS.md` с правилами (шаблон в [`templates/AGENTS.md`](../templates/AGENTS.md)).
4. Запусти `kimi` и опиши задачу.

**Что помнить:** Kimi уважает `AGENTS.md`. Если правила не написаны — агент угадывает.

---

## Claude Code

**Особенности:**
- Терминальный интерфейс.
- Мощные инструменты: bash, edit, view, grep.
- Хорошо работает с большими проектами.
- Естественный разговорный стиль.

**Как начать:**
1. Установи Claude Code.
2. Запусти `claude` в папке проекта.
3. Опиши задачу текстом.
4. Используй команды `/help` и `/clear`.

**Что помнить:** Claude Code любит обсуждать. Не бойся просить «короче» или «просто сделай».

---

## OpenCode

**Особенности:**
- Открытый AI-кодинг ассистент.
- Интегрируется с редакторами и CLI.
- Поддерживает MCP (Model Context Protocol) для расширения инструментов.

**Как начать:**
1. Установи OpenCode по официальной инструкции.
2. Настрой провайдера модели.
3. Открой проект и начни диалог.

**Что помнить:** Конфигурация гибкая, но требует внимания. Начни с простого сетапа.

---

## GitHub Copilot / Cody

**Особенности:**
- Copilot: встроен в VS Code, IDE JetBrains, Neovim.
- Cody (Sourcegraph): чат, команды, код-интеллект.
- Оба хороши для автодополнения и встроенного чата.

**Как начать:**
1. Установи расширение.
2. Авторизуйся.
3. Используй чат для вопросов и inline suggestions для мелких правок.

**Что помнить:** Copilot/Cody — скорее помощник в моменте. Для крупных задач используй полноценные агенты (Cursor, Kimi, Claude).

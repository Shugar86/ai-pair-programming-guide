# Примеры сессий

🌐 [English](../en/07-examples/README.md) · **Русский**

Этот раздел — не идеальные success story, а реальные диалоги с агентом. В каждой сессии есть момент, где что-то пошло не так: агент ушёл в сторону, забыл ограничение или предложил лишнее. Главное — как вернуть его на курс.

Если вы ещё не читали [`06-prompt-patterns.md`](../06-prompt-patterns.md), начните с него: примеры показывают, как эти паттерны работают в жизни.

## Как читать

Каждый файл построен по одной схеме:

1. **Задача** — что хотели получить.
2. **План** — что агент предложил до кода.
3. **Реализация и тупик** — где застряли и что сказали, чтобы выбраться.
4. **Проверка** — команды и тесты.
5. **Коммит** — итоговый diff.
6. **Рефлексия** — что сработало, а что стоит озвучить сразу.

# Пример: добавление фичи в Cursor

**Вердикт:** даже маленькая фича требует контроля scope.

**Среда:** Cursor  
**Задача:** Добавить кнопку «Копировать» рядом с email в профиле пользователя.

---

## 1. Определяем задачу

```text
Ты: Добавь кнопку "Копировать" рядом с email в профиле пользователя.
     При нажатии email копируется в буфер обмена.
     Текст кнопки меняется на "Скопировано" на 2 секунды, потом возвращается.
     Трогай только компоненты профиля.
```

## 2. Согласуем план

```text
Агент: План:
1. Найти компонент профиля, где отображается email.
2. Добавить кнопку Copy рядом с email.
3. Реализовать обработчик клика с navigator.clipboard.writeText.
4. Добавить data-testid для кнопки.

Ты: Ок. Без всплывающих уведомлений — только меняем текст кнопки.
```

## 3. Реализация и первый тупик

Агент предлагает diff. Ты видишь, что он зацепил 4 файла, включая глобальные стили.

```text
Ты: Стоп. Верни изменения только в ProfileCard.tsx и UserEmail.tsx. Остальное не трогай.
Агент: [откатывает лишние файлы]
```

Агент переписывает, но теперь кнопка добавлена в оба компонента — дублирование.

```text
Ты: Убери кнопку из ProfileCard. Оставь только в UserEmail.tsx — email там один, пусть логика живёт рядом с ним.
Агент: [переносит логику в UserEmail.tsx]
```

## 4. Проверка

```text
Ты: Запусти тесты для UserEmail.
Агент: npm test UserEmail
Агент: Тесты прошли, но один снапшот устарел.
Ты: Обнови снапшот, только если изменение в нём — это наша новая кнопка.
Агент: [обновляет снапшот и показывает diff]
Ты: Да, ок. Добавь data-testid="copy-email-button".
```

## 5. Коммит

```bash
git add src/components/UserEmail.tsx src/components/__tests__/UserEmail.test.tsx
git commit -m "feat: add copy email button in profile"
```

## 6. Рефлексия

- Сработало: план до кода выровнял ожидания. Ограничение «только компоненты профиля» помогло быстро отсечь лишнее.
- Первый тупик: агент зацепил глобальные стили. Checkpoint в Cursor спас от ручного отката.
- Можно лучше: сразу сказать «только UserEmail.tsx», чтобы не убирать дублирование во втором раунде.

## Мини-вывод

Даже простая кнопка может размазаться по проекту. Указывай scope до первого diff и читай каждое изменение. Cursor быстро исправляет ошибки направления, если ты ловишь их рано.

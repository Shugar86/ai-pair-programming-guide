# Пример: добавление фичи в Cursor

**Среда:** Cursor  
**Задача:** Добавить кнопку «Копировать» рядом с email в профиле пользователя.

---

## 1. Определяем задачу

```text
Ты: Добавь кнопку "Копировать" рядом с email в профиле пользователя.
     При нажатии email копируется в буфер обмена.
```

## 2. Согласуем план

```text
Агент: План:
1. Найти компонент профиля, где отображается email.
2. Добавить кнопку Copy рядом с email.
3. Реализовать обработчик клика с navigator.clipboard.writeText.
4. Добавить небольшое уведомление об успешном копировании.

Ты: Ок, но без уведомления — просто меняем текст кнопки на "Скопировано" на 2 секунды.
```

## 3. Реализация

Агент предлагает diff. Ты видишь, что он зацепил 2 лишних файла.

```text
Ты: Верни изменения в ProfileCard.tsx и UserEmail.tsx. Остальное не трогай.
```

Агент переписывает, оставляя только нужные файлы.

## 4. Проверка

```text
Ты: Запусти тесты для ProfileCard.
Агент: npm test ProfileCard
Ты: [смотрит diff] Выглядит ок. Добавь data-testid для кнопки.
```

## 5. Коммит

```bash
git add .
git commit -m "feat: add copy email button in profile"
```

## 6. Рефлексия

- Сработало: план до кода сразу выровнял ожидания.
- Можно лучше: сразу сказать «только 2 файла», чтобы не откатывать лишнее.
